## Descripción del Proyecto

Este proyecto consiste en la auditoría y rediseño de la aplicación Learn-Loop, una plataforma de micro-aprendizaje.

Se identificaron problemas como:

- Fricción negativa (notificaciones agresivas)  
- Sobrecarga cognitiva  
- Uso de dark patterns  
- Problemas de usabilidad  

### Objetivo

Crear una experiencia de aprendizaje:

- Motivadora  
- Justa  
- Clara  
- Centrada en el usuario
